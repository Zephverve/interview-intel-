# 报告输出模板

## 报告结构

```markdown
# {岗位} · 面经智能分析报告

> 生成时间 | 有效面经 N 篇 | 公司范围 | 简历：有/无

## 1–3. 概览 / 摘要 / 高频考点表

## 4. P0 · 必考（每题完整块）

## 5. P1 · 高频（每题完整块，格式同 P0，略短）

## 6. P2 · 差异化（每题完整块，非仅表格）

## 7. 分轮次面试预测
### 无简历：通用高频题列表
### 有简历：见第 8 节「分轮次个性化预测」

## 8. 简历交叉 + 分轮次个性化预测（有简历时）
### ✅ / ⚠️ / 💡
### 一面预测题 · 二面 · 三面 · HR（各 5–8 题 + 简历关联）

## 9. 7 天计划 · 10. 反问 · 11. 来源
```

## 单题输出模板（P0 / P1 / P2 均必用）

```
───────────────────────────────────────────────────
  P{N}-{序号} · {题目} `[标签]`
───────────────────────────────────────────────────

📊 出现 X 次 · {公司} · {类型}

📖 核心要点
① ...  ② ...  ③ ...

🗣️ 标准口语答案

"{完整口语段落 —— P0 约 8–15 句；P1 约 6–10 句；P2 约 4–6 句}"

🔍 追问方向
- ...
```

### 禁止

- P1 只写「标准答案摘要」
- P2 只在表格里写 2 句
- 有简历但分轮次预测仍用通用题、不绑项目
